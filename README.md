# 🧠 AI-Paper-Review

> "학부생의 시선으로 읽고 해석한 AI 논문 리뷰 모음집입니다."

이 저장소는 인공지능/딥러닝 분야의 주요 논문을 직접 읽고 요약하며,  
학부생 수준에서의 이해, 해석, 인사이트 를 축적하기 위해 만들어졌습니다.  
논문의 핵심 기여, 모델 구조, 실험 방법, 느낀 점 등을 직접 정리한 내용을 담고 있습니다.
